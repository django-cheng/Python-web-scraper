第一章：核心概念與基礎

Artificial Intelligence (AI)

人工智慧，電腦模擬人類智慧的技術總稱。
🔥 2048

Machine Learning (ML)

機器學習，讓電腦從數據中學習規律。

Deep Learning (DL)

深度學習，模仿人腦神經網絡的多層次學習。
廣告：想學 Python 嗎？點擊這裡！

Neural Network

類神經網絡，模擬生物神經元運作的數學模型。

Algorithm

演算法，解決特定問題的一系列計算步驟。

Data Science

資料科學，從數據中提取知識和洞察的學科。

Big Data

大數據，無法用傳統工具處理的海量資料。

Model

模型，訓練後用來預測或生成的數學表示。

Training

訓練，讓模型通過數據調整參數的過程。

Inference

推論，訓練好的模型對新數據進行預測。

Dataset

資料集，用於訓練或測試模型的數據集合。

Label

標籤，數據的正確答案或分類結果。

Feature

特徵，數據中對預測有幫助的屬性。

Accuracy

準確率，模型預測正確的比例。

Loss Function

損失函數，衡量模型預測與真實值的差距。

Optimizer

優化器，用於更新模型參數以減少錯誤的算法。

GPU

圖形處理器，加速深度學習運算的硬體。

CUDA

NVIDIA 推出的平行運算平台，用於加速 AI。

Python

目前 AI 領域最主流的程式語言。

TensorFlow

Google 開發的開源機器學習框架。

PyTorch

Meta 開發的深度學習框架，學術界常用。

Scikit-learn

Python 的傳統機器學習庫。

Pandas

Python 的資料分析與處理庫。

NumPy

Python 的數值計算基礎庫。

Matplotlib

Python 的資料視覺化繪圖庫。

第三章：神經網路與技術細節

ArXiv 論文網
  • #架構 Transformer (現代大型語言模型（如BERT, GPT）的基礎架構。)
  • #單位 Token (模型處理文字的最小單位，不完全等於單字。)
  • #參數 Parameter (模型內部可調整的變數，決定模型大小。)
  • #參數 Weights (權重，神經網絡中連接強度的數值。)
  • #參數 Bias (偏差，神經元激活的閾值偏移量。)
  • #訓練 Epoch (完整遍歷一次所有訓練數據的次數。)
  • #訓練 Batch Size (每次更新參數前處理的樣本數量。)
  • #訓練 Learning Rate (學習率，決定每次參數調整幅度的大小。)
  • #問題 Overfitting (過度擬合，模型死記硬背，對新數據表現差。)
  • #問題 Underfitting (低度擬合，模型沒學好，連訓練數據都預測不準。)
  • #函數 Activation Function (激活函數，如 ReLU，引入非線性特徵。)
  • #函數 Sigmoid (將輸出壓縮在 0 到 1 之間的激活函數。)
  • #函數 Softmax (將輸出轉換為機率分佈的函數。)
  • #機制 Attention Mechanism (注意力機制，讓模型專注於輸入的重要部分。)
  • #機制 Backpropagation (反向傳播，計算誤差並更新權重的核心算法。)
  • #機制 Gradient Descent (梯度下降，尋找損失函數最小值的優化方法。)
  • #預處理 Normalization (正規化，將數據縮放到特定範圍。)
  • #預處理 One-hot Encoding (獨熱編碼，將類別轉換為二進制向量。)
  • #評估 Confusion Matrix (混淆矩陣，視覺化分類模型表現的表格。)
  • #評估 Precision (精確率，預測為正樣本中真正為正的比例。)
  • #評估 Recall (召回率，實際為正樣本中被預測出來的比例。)
  • #評估 F1-Score (精確率與召回率的調和平均數。)
  • #技術 Dropout (隨機丟棄神經元，防止過度擬合的技術。)
  • #技術 Convolution (卷積運算，提取圖像特徵的核心操作。)
  • #技術 Pooling (池化，降低特徵維度並保留重要資訊。)

第四章：應用挑戰與術語字典

Hallucination
幻覺，AI 生成看似合理但完全錯誤的資訊。
Prompt Engineering
提示工程，設計輸入指令以引導 AI 輸出高品質結果。
Token Limit
模型一次能處理的最大文字長度限制。
AD: 機器學習實戰課程特價中
Zero-shot Learning
零樣本學習，模型沒看過範例就能完成任務。
Few-shot Learning
少樣本學習，給模型看幾個範例後完成任務。
Chain of Thought
思維鏈，引導模型逐步推理出答案的技巧。
RAG
檢索增強生成，結合外部知識庫來回答問題。
API
應用程式介面，讓程式之間交換資料的橋樑。
Open Source
開源，公開程式碼供大眾使用與修改。
Hugging Face
AI 界的 GitHub，分享模型與資料集的平台。
Kaggle
著名的數據科學競賽平台。
Colab
Google 提供的免費雲端 Jupyter Notebook 環境。
Jupyter Notebook
互動式程式設計環境，適合資料分析。
Bias (Ethics)
偏見，AI 模型反映出人類社會的歧視或不公。
Black Box
黑盒子，難以解釋 AI 模型內部如何做出決策。
Explainable AI
可解釋 AI，致力於讓 AI 決策過程透明化。
Turing Test
圖靈測試，判斷機器是否具有人類智慧的測試。
Singularity
奇點，人工智慧超越人類智慧的時間點。
Data Augmentation
資料增強，透過旋轉、裁切等增加訓練數據。
Hyperparameter
超參數，訓練前需人工設定的參數（如學習率）。
Regression
迴歸，預測連續數值（如房價）的任務。
Classification
分類，預測類別（如貓或狗）的任務。
Clustering
分群，將相似的數據歸類在一起。
Dimension Reduction
降維，減少資料特徵數但保留重要資訊。
IoT
物聯網，常與 AI 結合成為 AIoT。
